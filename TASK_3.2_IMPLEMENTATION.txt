USAGE IN TEMPLATES:
===================

In front-page.php and other templates:

<?php
$homepage_mgr = new Airlinel_Homepage_Manager();

if ($homepage_mgr->get_section_visibility('featured_routes')) {
    $content = $homepage_mgr->get_section_content('featured_routes');
    ?>
    <section class="featured-routes">
        <?php echo $content ? wp_kses_post($content) : airlinel_get_default_featured_routes(); ?>
    </section>
    <?php
}
?>

DATABASE STORAGE:
=================

Each section stores two wp_options entries:

- airlinel_homepage_section_{id}_visible (boolean)
- airlinel_homepage_section_{id}_content (text/HTML)

Example:
- airlinel_homepage_section_featured_routes_visible = 1
- airlinel_homepage_section_featured_routes_content = "<p>Custom content here...</p>"
