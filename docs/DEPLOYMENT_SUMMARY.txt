1. SSH/FTP into server

2. Navigate to WordPress theme directory:
   cd wp-content/themes/airlinel-transfer-services/

3. Upload all .mo files to languages/ directory:
   Upload: airlinel-theme-*.mo files

4. Replace theme files in root directory:
   Replace: header.php, page-booking.php, functions.php

5. Replace JavaScript files:
   Replace: assets/js/header-selectors.js, assets/js/booking.js

6. Replace language settings:
   Replace: includes/class-language-settings.php

7. Set permissions:
   chmod 644 languages/*.mo
   chmod 644 header.php
   chmod 644 page-booking.php
   chmod 644 functions.php
   chmod 644 assets/js/header-selectors.js
   chmod 644 assets/js/booking.js
   chmod 644 includes/class-language-settings.php

8. In WordPress Admin:
   - Go to Settings → General
   - Change "Site Language" to Turkish (Türkçe) or desired language
   - Click "Save Changes"
   - Clear cache if available

9. Clear browser cache and test
